## 介绍

仿微信二维码扫描，中间扫描区域透明效果。界面主要`CoreGraphics`绘制。



##版本
*	1.0.0 实现二维码中间扫描区域的透明的主要功能。

*	1.0.1 把扫描区域限制在中间透明的范围。

*   1.0.2 增加扫描的类型，支持二维码和条形码扫描。

*	1.0.3 增加摄像头有效性的判断，二维码只能在真机测试。

*	1.0.4 修改iPad横屏不能扫描的bug

*	1.0.5 重构部分代码

*	1.0.6 push一个界面再返回，界面卡住问题

*	1.0.7 增加二维码图片的读取	

* 1.0.8 修正相机权限
